# Separate Invoice by Company Code and Year

## 📌 Description
This project separates Invoices by Company Code and by Year, then move them into dedicated folders.

## Activities Used
- Invoke workflow file
- Assign
- For each file in Folder
- Switch
- Move File

## Explanation
- Created two separate xaml files to separate Invoices by Company code and by year
- used Invoke Workflow file to invoke those two xaml files in main file
